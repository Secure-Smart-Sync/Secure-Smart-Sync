### What? 

The `Engineering` folder contains all the relevent documentations regarding the Technical and problem solving aspect of this obsidian plugin, detailing decisions made throughout the process and reasoning behind those, along with compromises and tradeoffs made in the architecture for the sake of things we consider 'uncompromizable'.

## what is sss?
Secure-Smart-sync is basically a obsidian plugin that syncs your vault across devices.

## why? 
because sync is a very useful feature that allows you to write from anywhere and continue it without much thinking. there is [official obsidian sync](https://obsidian.md/sync), which is fast, reliable, and possibly secure.
