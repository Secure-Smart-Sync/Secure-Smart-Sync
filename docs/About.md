# About

Secure Smart Sync (SSS) is an open-source project created by an independent solo developer, **Sen**, as a free and privacy-focused alternative to Obsidian Sync.

The project was built to give users full ownership over their notes by allowing them to sync their Obsidian vaults through their own Cloudflare R2 storage while keeping files encrypted locally before upload. The goal has always been simple: provide a reliable sync solution without subscriptions, vendor lock-in, or compromising user privacy.

Development of SSS was accelerated with the help of agentic AI tools for rapid iteration, debugging, and implementation support. The project itself followed an agile and highly iterative development process, where features were continuously refined based on architectural constraints, usability issues, and edge cases discovered during testing.

The initial public release of **SSS v1.0.0** was reached after **26 major iterations** of the codebase before it was considered stable enough for release.

All branding assets—including the project name, logos, icons, and visual identity—were created by Sen and remain exclusive to this project.
